## Running Solutions
To compile and run any C++ solution:
```bash
g++ -o solution ProblemName.cpp
./solution < input.txt
```

Or for online judges:
```bash
g++ -std=c++17 -O2 -Wall ProblemName.cpp -o ProblemName
```
